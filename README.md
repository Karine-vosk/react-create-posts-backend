## Project Structure

This project consists of two repositories:

### Frontend
React app for creating and displaying posts  
Repo: https://github.com/Karine-vosk/react-create-posts

### Backend
Node.js + Express REST API  
Repo: https://github.com/Karine-vosk/react-create-posts-backend

## How to run locally
### Frontend
npm install
npm start

### Backend
```bash
